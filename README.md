
# 🎮 Simon Game

A simple and interactive memory-based game built using HTML, CSS, JavaScript, and jQuery.  
The game challenges players to remember and repeat an increasing sequence of colors.

---

## 🚀 Features
- Random color sequence generation
- User interaction through button clicks
- Sound effects for each color
- Level progression system
- Game over detection with restart functionality
- Smooth animations using jQuery

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- jQuery

---

## 🎯 How to Play
1. Press the **"A" key** to start the game.
2. Watch the pattern shown by the system.
3. Repeat the sequence by clicking the colored buttons.
4. Each level adds a new step to the sequence.
5. If you click the wrong button, the game ends.
